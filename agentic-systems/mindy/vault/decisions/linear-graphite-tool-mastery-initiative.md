# Linear and Graphite Tool Mastery Initiative

*Decision made: 2026-04-30*

## Context
Team recognized need for enhanced project management capabilities and development workflow optimization to support growing complexity and coordination requirements across multiple concurrent projects.

## Decision
Implement comprehensive tool mastery initiative focusing on Linear and Graphite adoption:
- Mandatory team training on Linear for project management including initiatives, projects, and issue tracking
- Graphite adoption for stacked PR workflow and improved code review processes
- Structured approach to project organization with clear distinctions between initiatives and projects
- Enhanced labeling and filtering capabilities for improved workflow management
- Video tutorial assignments with Friday deadline for foundational understanding
- Focus on mastering tools for deep system architecture and AI-driven development workflows

## Rationale
- Improves project visibility and coordination across multiple concurrent efforts
- Enables better tracking of initiatives versus shorter-term projects
- Reduces merge conflicts and integration complexity through stacked PR workflows
- Supports transition from manual coding to system-level mastery and AI-driven development
- Enhances team coordination and communication through standardized tooling
- Provides foundation for scalable development processes as team and project complexity grows

## Implementation
- Team members assigned Linear and Graphite tutorial videos with Friday completion deadline
- Project structure reorganization in Linear with clear initiative and project distinctions
- Graphite integration for stacked PR workflow implementation
- Enhanced labeling and filtering processes for improved workflow management
- Regular review and refinement of tool usage for continuous optimization
